​​​​​​​Create a method that sums 1,…,n numbers as parameters and print the operation result.

```
 // Input arguments.
 2,4,5,8,10
 
 // Output. A string that contains the operation result in this format.
 2 + 4 + 5 + 8 + 10 = 29
```
